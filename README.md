# ApiPokemon
Api Rest de Pokémon hecha con Spring Boot

## Funcionalidades

- Devolver archivo .JSON con Nombre, Tipo y Imagen del Pokémon (URL)
- Poder pedir atributo directamente vía flags (Ej: API/ID/TIPO)
